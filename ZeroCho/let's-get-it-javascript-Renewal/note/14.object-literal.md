# object literal (객체 리터럴)

- 배열이나 함수가 아닌 특정 모양을 가지는 경우

```
const apple = "사과";
const banana = "바나나";
const melon = "메론";
```

```
const fruit = ["사과","바나나","메론"];
```

- 위 코드를 아래와 같이 객체 리터럴로 표기 해줄 수 있다.
  - 여러가지 속성들을 값만 필요하다면 배열이 좋지만,

```
const fruit = {
  apple : "사과",
  banana : "바나나",
  melon : "메론",
}
```
