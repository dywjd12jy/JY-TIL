# filter() ,substring()

- filter()는 배열에서 사용가능
  `"abc".filter()` // error
  `"abc".split().filter()` - 이렇게 사용해야 함 - String.prototype.split()
  `"abc".split() // ["a","b","c"]` - split 함수 설명

- substring()는 string에서 사용가능
  -- 문자열에서 부분문자열로 반환해준다.
  -- substring() 와 substr() 구분 잘 하기
  `String.prototype.substring() `
